# What the app does

Status: spec

* the user is a female who wants to keep track of her period
* the user puts in a date when the period first arrived
    * some initial date is set when the app is first opened
    * the date can be adjusted anytime
* anytime the user opens the app ...
    * there is immediate visual feedback where she is on the period calendar
        * e.g. green: day without period
        * yellow: period may arrive the following day
        * red: period is due
* daily in the morngin, the user gets a push notification informing on the current state, color-coded
    * this can be configured, e.g.
        * time of the notification
        * inform daily, or inform only status yellow and red, or inform only status red
